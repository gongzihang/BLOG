📘 Markdown 写作模板（含 Front Matter 与语法指南）

第一部分：📄 Front Matter 配置说明

Front Matter 是 Markdown 文件顶部的元信息，用于配置文章标题、标签、分类、日期等内容。通常写在 --- 包裹的 YAML 格式中。

🧾 示例模板

📋 字段说明表

字段名 说明 默认值
layout 页面布局模板（如 post、page） config.default_layout
title 页面标题 文件名
date 建立日期 文件创建时间
updated 最近更新日期 文件更新时间
comments 是否启用评论 true
tags 标签数组，用于分类内容 空
categories 分类数组（适用于归档） 空
permalink 自定义永久链接（应以 / 或 .html 结尾） null
excerpt 页面摘要，建议结合插件处理 空
disableNunjucks 禁用 Nunjucks 模板语法（{{ }}、{% %}） false
lang 页面语言，覆盖默认配置 继承自 _config.yml
published 是否发布该文章 _posts 下为 true，草稿为 false
